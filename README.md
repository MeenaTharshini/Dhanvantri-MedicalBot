# 🌿 Dhanvantri AI

Dhanvantri AI is a chatbot that provides traditional health guidance based on:

- Ayurveda
- Siddha medicine
- Herbal remedies
- Yoga suggestions

## Features

- Symptom-based remedies
- Ayurveda knowledge database
- Siddha terminology explanations
- Daily health tips
- Yoga posture suggestions

## Technologies Used

- Python
- Flask
- HTML
- CSS

## Run the Project

```bash
pip install -r requirements.txt
python app.py
